Installs options for the Maxis Highway networks (Highway and Ground Highway).  Default Style keeps Maxis Highways in their original state.  Maxis Highway Override changes the Maxis Highways to look more like the NAM's RealHighway (RHW) network, which uses the RHW's Modular Interchange System (MIS) functionality and its own interchanges, and is NOT compatible with the game's original Maxis Highway interchanges.
